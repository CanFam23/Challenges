# Challenges

This repository contains several challenges meant to practice various coding skills

## Java Setup & Running the Project

### Prerequisites

Make sure you have **Java 21+** installed (not tested with other versions).

Check your Java version:

```bash
java -version
```


### Clone the Repository

```bash
git clone https://github.com/CanFam23/Challenges.git
cd Challenges
```

---

### Running each challenge

To run each challenge, navigate to the intended subdirectory
and follow the instructions in its README.
